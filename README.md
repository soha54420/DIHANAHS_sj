## 🌸 Strugglehs

If you're here because Git is throwing weird errors at you, welcome. You're not the first person and definitely won't be the last.

At some point during this project, Git told me things like:

* `remote origin already exists`
* `src refspec master does not match any`
* `non-fast-forward`
* and then Vim appeared out of nowhere.

For a while, none of it made sense.

The funny thing is that most Git errors sound way scarier than they actually are. Usually, Git is trying to tell you what's wrong, just in a language that feels impossible to understand when you're new.

So if you're stuck, don't rush, don't start randomly copying commands from the internet, and definitely don't panic.

Read the error once more. Search it. Ask someone. Try again.

Most problems end up teaching you something you'll remember forever.

And if Vim suddenly opens and you have no idea what happened...

`:wq`

Trust me.
